Private LLM — API 설정

OpenAI 호환 LLM 서버 연결 정보를 입력하세요.

서버 주소를 입력하세요. /v1로 끝나면 그대로 사용, 아니면 자동으로 /v1을 추가합니다.
예: http://llm-server:8000  /  http://llm-server/api
API Key는 브라우저 로컬 스토리지에만 저장되며, 외부로 전송되지 않습니다.
서버에서 제공하는 모델 이름 (기본값: gpt-3.5-turbo)
100~32000 (기본값: 2000)

사용 방법

  1. 내부 LLM 서버의 Base URL을 입력합니다.
  2. API Key를 입력합니다 (서버가 키를 요구하지 않아도 임의 값 입력).
  3. 모델 이름을 입력합니다.
  4. 저장 후 연결 테스트로 확인합니다.
  5. 브라우저 팝업에서 요약 / Q&A / 내보내기를 사용합니다.
🔒 API Key는 chrome.storage.local에 저장되며 해당 브라우저 프로필에만 접근 가능합니다. 외부 서버로 전송되지 않습니다.
